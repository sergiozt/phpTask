Предложения по улучшению запроса:

Показатель в скорости выборки при использовании явных Join-ов выше чем у неявных Join-ов.
SELECT * FROM data LEFT JOIN link ON data.id = link.data_id LEFT JOIN information ON link.info_id = information.id


Предложения по улучшению таблици:

Так как нет информации о том, что именно будет храниться в таблицах(не ясно предназначение поля value в таблице data)б
можно предположитть что оно не нужно а если у нужно(напрмиер для статуса: 0 или 1)тогда тип поля не int а tinyint ну или
точно зная структура то можно и set.
Также для таблици link для ее полей установил(в приклепреллном файе видно) можно установить Foreign Keys, что бы при
удалении записи из одной из таблиц (data, information) поля с таблици с соолтвествующими значениями id удалялись с таблици
data.

Сложно судить об возможных улычшениях, не имея полную картину предназначения данной базы.
